This is Android driver for OMNIKEY readers.

Release 1.7.12.0: 


This release provides the following bugfixes and improvements:
    

==============================================================
* Added USB_DEVICE_ATTACHED intent


Files
=====

* CardReaderManager-release.apk - card reader manager
* jsr268library.jar - JSR268 Java library
* JSR268Client.apk - installation package for sample client application
* JSR268Client.zip - source code for sample application
* Documentation\PLT-02493_A.0_OMNIKEY_Android_Driver_Installation_Guide_en.pdf - installation guide
* Documentation\PLT-02492_1.4_OMNIKEY_Android_Driver_User_Guide_en.pdf - user guide
* README.txt - this file


Supported readers
==================

* smart@link chipset
* OMNIKEY 1021 (USB)
* OMNIKEY 3021 (USB)
* OMNIKEY 3121 (USB)
* OMNIKEY 3621 (USB)
* OMNIKEY 3821 (USB)
* OMNIKEY 6121 (USB)
* OMNIKEY 2061 (Bluetooth)
* OMNIKEY 5427 (USB)
* OMNIKEY 5022 (USB)
* OMNIKEY 5127 (USB)
* OMNIKEY 5127 Mini (USB)


Quick start
===========

1. Install CardReaderManager.apk on Android device
2. Create your Android application and link with jsr268library.jar
3. Install your application

Quick start for the demo app
============================
1. Install CardReaderManager.apk on Android device
2. Install  JSR268Client.apk

For more details see PLT-02492_1.4_OMNIKEY_Android_Driver_User_Guide_en.pdf
